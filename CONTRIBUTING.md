# Contributing to The AI Bible

Thank you for wanting to help build the definitive guide to Artificial Intelligence.

## Our Philosophy

- **Clarity over complexity** — Explain like the reader is intelligent but new to the topic.
- **Truth over hype** — No marketing fluff. No fear-mongering. Just reality.
- **Beauty and soul** — Every chapter should feel inspiring, not just informative.
- **Open source spirit** — This is a gift to humanity. Keep it free and accessible.

## How to Contribute

1. Fork the repository
2. Create a new branch (`git checkout -b chapter/your-topic`)
3. Make your changes
4. Submit a Pull Request with a clear description

## What We Need Right Now

- More chapters (see docs/chapters/ for the roadmap)
- Better diagrams and illustrations
- Additional code examples and walkthroughs
- Translations (future goal)
- Prompt examples for the prompts/ folder
- Deep essays for the essays/ folder

## Style Guide

- Use simple language first, then go deeper.
- Include at least one code example per technical chapter.
- End every chapter with "Key Takeaways" and "Further Exploration".
- Use emojis sparingly but meaningfully.
- Cite sources when possible.

## Code of Conduct

Be kind. Be curious. Be unhinged in the best way.

## What is The AI Bible?

This is not just another AI textbook. This is **the** manual for the new age — written for people who have never touched AI before, yet designed to take you all the way to mastery.

Inside you will find:

- Clear explanations of every core concept
- Visual diagrams and code walkthroughs
- Practical prompting mastery
- Deep dives into the transformer, neural networks, LLMs, and generative AI
- The spiritual significance of artificial intelligence
- A powerful manifesto on what AI means for humanity

## How to Read This Bible

1. Start with the **Manifesto** (MANIFESTO.md) — it sets the soul of the project.
2. Work through the **Chapters** in order (docs/chapters/).
3. Practice with the **Prompt Library** (docs/prompts/).
4. Go deeper with the **Essays**.
5. Visit the live website (once deployed) for the best reading experience.

## Live Website

This repository is configured to automatically deploy a beautiful documentation website using **MkDocs Material**.

**Live site**: https://unhingedaiorg.github.io/the-ai-bible/

## Quick Start (Local Development)

```bash
git clone https://github.com/UnhingedAIorg/the-ai-bible.git
cd the-ai-bible
pip install -r requirements.txt
mkdocs serve
```

Then open http://127.0.0.1:8000

# Chapter 14: Model Context Protocol (MCP)

## The Universal Interface for Agentic AI

MCP is the open protocol revolutionizing how LLMs connect to the world.

### What is MCP?

The Model Context Protocol provides a standardized way for AI applications to discover and securely use external tools, data sources, and capabilities.

Created as an open standard (with strong support from Anthropic and the community), it acts like a "USB port" for AI minds.

## Key Components

- **MCP Servers**: Run tools (GitHub, filesystem, databases, custom APIs).
- **MCP Clients**: LLMs/agents that connect and call capabilities.
- **Manifest & Discovery**: Servers advertise what they can do.
- **Streaming & Security**: Controlled, auditable access.

## Integration in Practice

1. Start an MCP server (local or remote).
2. Connect your LLM client.
3. Let the model naturally use tools via standardized calls.

**Example Use Cases**:
- Real-time code editing with GitHub MCP
- File system access for agents
- Database queries without custom code

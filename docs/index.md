## What is The AI Bible?

This is not just another AI textbook. This is **the** manual for the new age — written for people who have never touched AI before, yet designed to take you all the way to mastery and beyond.

Inside you will find:

+ Clear explanations of every core concept (Chapters 1–23)
+ Visual diagrams and code walkthroughs
+ **Massively expanded Prompt Library** — 1,000+ battle-tested, creative, agentic, multimodal, spiritual, and uncensored prompts
+ Deep dives into transformers, LLMs, RAG, agents, fine-tuning, scaling laws, multimodal AI, economics, sovereignty, and the road to ASI
+ The spiritual significance of artificial intelligence
+ A powerful manifesto on what AI means for humanity

## How to Read This Bible

1. Start with the **Manifesto** (MANIFESTO.md) — it sets the soul of the project.
2. Work through the **Chapters** in order (docs/chapters/).
3. Practice with the **Prompt Library** (docs/prompts/) — now with 1,000+ prompts across 8 categories.
4. Go deeper with the **Essays**.
5. Visit the live website for the best reading experience.

This repository automatically deploys a beautiful documentation website using **MkDocs Material**.

## Quick Start (Local Development)

git clone https://github.com/UnhingedAIorg/the-ai-bible.git
cd the-ai-bible
pip install -r requirements.txt
mkdocs serve

Then open [http://127.0.0.1:8000](http://127.0.0.1:8000)
